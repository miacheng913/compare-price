# 比價小幫手

一個為手機採買情境設計的價格紀錄與比較工具。資料保存在瀏覽器本機，不需註冊帳號，也不會上傳購物紀錄。

## 功能

- 商品、品牌與比價群組管理
- 各賣場價格、數量、日期與備註紀錄
- 自動換算每單位價格並標示最低價
- 各賣場最新價格與完整價格歷史
- 商品搜尋、分類篩選、賣場與分類管理
- 深色模式與手機安全區支援
- iOS 主畫面專屬 App Icon
- GitHub Pages 自動部署

## 本機執行

需要 Node.js 22 與 pnpm。

```bash
pnpm install
pnpm dev
```

正式建置：

```bash
pnpm build
```

靜態網站會輸出到 `out` 資料夾。
